Word Engine

Write a C program that acquires free text from the keyboard, consisting of several lines (max 1000) of a number of characters not exceeding 100 each.
The insertion ends when the user inserts a row equal to END.
At the end of the text acquisition, the program must print the following statistics: 1.
the total number of rows inserted; 2.
the total number of characters entered; 3.
the total number of alphanumeric characters entered; 4.
the total number of words entered.
Example: This is a test! To prove that my program actually works! END Total rows = 3 Total Char = 54 Total AlphaNumerical char = 44 Total words = 11

Input

The first N lines contain parts of a string A.

Output

The first line contains the total number of rows (excluded last row with END).
The second line contains the total number of characters.
The third line contains the total number of alphanumeric characters.
The forth line contains the total number of words.
