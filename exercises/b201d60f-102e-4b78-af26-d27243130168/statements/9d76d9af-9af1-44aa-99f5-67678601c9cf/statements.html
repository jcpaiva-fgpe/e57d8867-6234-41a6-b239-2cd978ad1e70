Word Length

Your task is to write a function count_length that takes in input an array A of words and counts for each word its length.

Input

The first line of input contains some words separed by whitespace

Output

The output displays the length of each word inside the array A.

Sample Input Sample Output
Milano Roma Firenze Napoli 6 4 7 6
Education Programming FGPE Happiness 9 11 4 9
Music Sport Art 5 5 3
