Account encrypting

Your task is to write a function encode64 that encode a string in base64 utf-8.
In particular you have to code an account information (username U and password P).
First thing you must chose a special symbol X from the above: + - & | ! ( ) } { [ ] ^ ~ * ? : .
, < > _ / Then you have to read username U and password P.
You have to pass X,U,P to the function encode64.
The function will concatenate the strings in a string S with structure: Username + special_symbol + password Finally you have to encrypt string S in base64 utf-8 codification.
Tips.
You can work with the library base64.

Input

The first line of input contains a character X that is the special symbol.
The second line of input contains a string U that is the username.
The third line of input contains a string P that is the password.

Output

The first line displays the base64 utf-8 encrypted string S if the requirements are fulfilled, Wrong otherwise.
