Crittografia dell'account

Il tuo compito è scrivere una funzione encode64 che codifichi una stringa in base64 utf-8. In particolare è necessario codificare le informazioni di un account (nome utente U e password P). Per prima cosa devi scegliere un simbolo speciale X tra quelli sopra: + - & | ! ()} {[] ^ ~ *? :. , <> _ / Quindi devi leggere username U e password P. Devi passare X, U, P alla funzione encode64. La funzione concatenerà le stringhe in una stringa S con struttura: Username + special_symbol + password Infine devi crittografare la stringa S nella codifica base64 utf-8. Suggerimenti. Puoi lavorare con la libreria base64.

Input

La prima riga di input contiene un carattere X che è il simbolo speciale. La seconda riga di input contiene una stringa U che è il nome utente. La terza riga di input contiene una stringa P che è la password.

Output

La prima riga mostra la stringa crittografata base64 utf-8 S se i requisiti sono soddisfatti, altrimenti "Wrong".
