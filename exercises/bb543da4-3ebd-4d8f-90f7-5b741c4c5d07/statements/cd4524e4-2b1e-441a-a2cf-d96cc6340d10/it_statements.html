Ordina i numeri in ordine crescente

Il tuo compito è scrivere un metodo che accetta un array di numeri interi e restituisce un nuovo array, ordinato in ordine crescente (dal più piccolo al più grande).
1. Ordina l'array di numeri interi in ordine crescente.
2. Se l'argomento della funzione è null, un array vuoto o non definito; restituisce un array vuoto.
3. Restituisce un nuovo array di numeri ordinati.
Note L'ingresso del test può essere positivo o negativo.

Input

La prima riga di input contiene la lunghezza dell'array.
La seconda riga di input contiene gli elementi dell'array.

Output

La prima riga contiene la matrice ordinata in ordine crescente.

Sample Input Sample Output
4
12 3 1 -9		 [-9, 1, 3, 12]
0 []
7
-2 4 1 3 12 -3 5		 [-3, -2, 1, 3, 4, 5, 12]
