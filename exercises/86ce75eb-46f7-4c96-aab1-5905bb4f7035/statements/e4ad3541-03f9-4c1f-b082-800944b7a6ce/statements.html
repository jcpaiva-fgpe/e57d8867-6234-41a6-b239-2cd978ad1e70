Absolute values sum

Your task is to write a short program that takes in input N numbers (positive and negative) and sums their absolute value.
Tips: An Absolute value of a number is the number without its sign (positive or negative)

Input

The first line contains a number N that is the expected numbers to take in input.
The second line contains all the N numbers divided by space “ “.

Output

The output contains the sum of all numbers given in input

Sample Input Sample Output
3
1 2 3		 6
4
-34 5 -44 3		 86
3
23 -44 3		 70
