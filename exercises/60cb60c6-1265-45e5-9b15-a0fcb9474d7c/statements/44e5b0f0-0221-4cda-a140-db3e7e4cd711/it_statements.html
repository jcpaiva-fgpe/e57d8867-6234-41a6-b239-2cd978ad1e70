Questa parola è palindromo?

Il tuo compito è leggere in input una stringa e controllare se è una parola palindromo. Una parola palindromo è una parola che si legge come avanti e indietro (es. Anna, Racecar, Madam)

Input

La prima riga di input contiene una stringa A.

Output

La prima riga viene visualizzata se la stringa A è palindromo o meno.

Esempio di input Esempio di output
Telefono No
Auto da corsa Sì
Anna Sì
