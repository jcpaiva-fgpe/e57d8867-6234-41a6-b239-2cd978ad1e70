Is this word palindrome?

Your task is to read in input a string and check if it is a palindrome word. A palindrome word is a word which reads the same backward as forward (eg. Anna, Racecar, Madam)

Input

The first line of input contains a string A.

Output

The first line displays if the string A is palindrome or not.

Sample Input Sample Output
Phone No
Racecar Yes
Anna Yes
