String Comparison

Write a function in C, called starter, which evaluates how many leading characters are in common between two given strings A and B.
The function takes two parameters, both of type string, and returns the integer N.
For example: • if the function were called as starter("hello", "he"), it should return 2 as the first two characters are identical.
• if the function were called as starter("hello", "yes"), it should return 0 as no leading character is in common

Input

The first line contains the string A.
The second line contains the string B.

Output

The output contains the integer N.

Sample Input Sample Output
Hello
He		 2
FGPE
GP		 0
Programming
Pro		 3
