Futuristic Language

Write a program that takes an English phrase and translate its in a futuristic language based on those rules: Replace each char with: • S to 5 • A to 4 • O to 0 (zero) • E to 3 • B to 8 • I to 1

Input

The first line contains a string A.

Output

The first line contains the string B with a futuristic language.

Sample Input Sample Output
FGPE FGP3
Hello! My name is John H3ll0! My n4m3 15 J0hn
Hello World! H3ll0 W0rld!
