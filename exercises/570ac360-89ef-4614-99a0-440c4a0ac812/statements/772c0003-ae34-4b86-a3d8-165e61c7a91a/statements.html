Word in list

Your task is to write a function “Is_in” that takes in input a word and a list of words and checks if the word is in the list.

Input

The first line of input contains a number N (N<10), the size of list L.
The second line of input contains the N words (separated by space) of the list L.
The third line of input contains the word W to search in the list L.

Output

The first line displays “Yes” if the word W is present in the list L, “No” otherwise.

Sample Input Sample Output
3
dog cat mouse
dog		 Yes
4
doll sun heart love
cat		 No
5
cell phone printer computer alexa
cellphone		 No
