Parola nell'elenco

Il tuo compito è scrivere una funzione "Is_in" che prenda in input una parola e un elenco di parole e controlli se la parola è nell'elenco.

Input

La prima riga di input contiene un numero N (N <10), la dimensione della lista L.
La seconda riga di input contiene le N parole (separate da spazio) della lista L.
La terza riga di input contiene la parola W a cerca nell'elenco L.

Output

La prima riga visualizza "Sì" se la parola W è presente nell'elenco L, "No" altrimenti.

Sample Input Sample Output
3
dog cat mouse
dog		 Yes
4
doll sun heart love
cat		 No
5
cell phone printer computer alexa
cellphone		 No
