Isogrammi

Il tuo compito è scrivere una funzione che, data una parola come input, determini se si tratta di un isogramma. Un isogramma (noto anche come "parola senza schema") è una parola o una frase senza che le lettere siano ripetute, tuttavia gli spazi e i trattini possono apparire più volte.

Input

La prima riga di input contiene una parola A.

Output

La prima riga mostra "Vero" se la parola è un isogramma, "Falso" in caso contrario.

Sample Input Sample Output
lumberjacks True
background True
isograms False
