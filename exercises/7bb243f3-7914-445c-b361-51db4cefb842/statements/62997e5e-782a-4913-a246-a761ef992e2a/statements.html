Isograms

Your task is to write a function that, given a word as input, determine if it is an isogram.
An isogram (also known as a "nonpattern word") is a word or phrase without a repeating letter, however spaces and hyphens are allowed to appear multiple times.

Input

The first line of input contains a word A.

Output

The first line displays “True” if the word is an isogram, “False” otherwise.

Sample Input Sample Output
lumberjacks True
background True
isograms False
