Is Johnny Making Progress?

To train for an upcoming marathon, Johnny goes on one long-distance run each Saturday.
He wants to track how often the number of miles he runs this Saturday exceeds the number of miles run the previous Saturday.
This is called a progress day.
Your task is to write a function that takes in an array of miles run every Saturday and returns Johnny's total number of progress days.

Input

The first line of input contains the array.

Output

The first line contains the total number of progress days.

Sample Input Sample Output
3 4 1 2 2
10 11 12 9 10 3
6 5 4 3 2 9 1
