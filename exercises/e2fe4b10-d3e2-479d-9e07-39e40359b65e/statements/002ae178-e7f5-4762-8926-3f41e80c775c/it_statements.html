Stringhe simili

Scrivete un programma che, dopo aver letto due stringhe dalla tastiera, visualizzi il messaggio
EQUALS se le due stringhe sono uguali,
SIMILAR se le stringhe differiscono solo per l'uso di lettere minuscole e maiuscole,
DIFFERENT altrimenti
(le stringhe possono contenere spazi).

Input

La prima riga contiene la stringa A.
La prima riga contiene la stringa B.

Output

La prima riga mostra il messaggio. (EQUALS, SIMILIAR, DIFFERENT)

Sample Input Sample Output
FGPE
fgpe		 SIMILAR
This is a test
THIS IS		 DIFFERENT
Hello world!
Hello world!		 EQUALS
