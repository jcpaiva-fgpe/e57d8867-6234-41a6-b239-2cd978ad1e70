Similar strings

Write a program which, after reading two strings from the keyboard, displays the EQUAL message if the two are strings are the same, SIMILAR if the strings differ only in the use of lowercase letters and uppercase, DIFFERENT otherwise (strings can contain spaces).

Input

The first line contains the string A.
The first line contains the string B.

Output

The first line displays the message.
(EQUAL, SIMILAR, DIFFERENT)

Sample Input Sample Output
FGPE
fgpe		 SIMILAR
This is a test
THIS IS		 DIFFERENT
Hello world!
Hello world!		 EQUALS
