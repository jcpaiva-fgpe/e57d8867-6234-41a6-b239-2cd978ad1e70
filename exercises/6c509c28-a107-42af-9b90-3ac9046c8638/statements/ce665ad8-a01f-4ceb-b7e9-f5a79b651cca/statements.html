Grand Gala

The prince Michael of a very, very distant kingdom is organizing a grand gala and would like the guests to be checked at the entrance.
In particular, he sent a list I with the names of the people invited and a list U of absolutely unwelcomed people.
Given a name N, the bouncer must decide whether to allow access or not and if call guards.
Write a function check_in(guest) that does this job.
The invited list is: I = [ LARRY, MEGAN, CONSUL, NOTARY, GOBLIN, CIRO ] The UNWANTED list is: U = [ VAMPIRE, SKELETON, DRAKE, VOLDEMORT]

Input

The first line contains the name N of the person who would like to join party

Output

The first line contains: OK if in list I.
NOT IN LIST if not in list I.
CALL GUARDS! If in unwanted list U.

Sample Input Sample Output
sara NOT IN LIST
CIRO OK
voldemort CALL GUARDS!
