Numeri narcisistici

Un numero è narcisistico quando la somma delle sue cifre, con ciascuna cifra elevata alla potenza della quantità di cifre, è uguale al numero stesso.
153 -> 3 cifre -> 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153 -> Narcisistico
84 -> 2 cifre -> 8^2 + 4^2 = 64 + 16 = 80 -> Non narcisistico
Il tuo compito è scrivere un programma che, dato un numero intero positivo n, restituisca vero se il numero è narcisistico e falso se non lo è.

Ingresso

La prima riga di input contiene un numero intero positivo n.

Uscita

La prima riga contiene "true" o "false".

Input di esempio Output di esempio
153 vero
84 Falso
8208 vero
