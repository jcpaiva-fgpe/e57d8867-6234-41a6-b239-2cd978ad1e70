Password Check

Your task is to write a function check_password that checks if a given password is secure enough.
A password to be strong, must have: • At least ONE uppercase letter.
• At least ONE number.
• At least ONE special symbol.
• More than 8 characters.
Special symbols are: + - & | ! ( ) } { [ ] ^ ~ * ? : .
, < > _ /

Input

The first line of input contains a string password

Output

The first line displays True if password is strong, False otherwise.

Sample Input Sample Output
20Millions! True
30,20,10 False
FGPE_2020? True
