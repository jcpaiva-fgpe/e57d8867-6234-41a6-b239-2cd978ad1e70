Controllo password

Il tuo compito è scrivere una funzione check_password che controlli se una data password è abbastanza sicura.
Una password per essere complessa deve contenere:
• Almeno UNA lettera maiuscola.
• Almeno un numero.
• Almeno UN simbolo speciale.
• Più di 8 caratteri.
I simboli speciali sono: + - & | ! ()} {[] ^ ~ *? :. , <> _ /

Input

La prima riga di input contiene una stringa password

Output

La prima riga mostra True se la password è forte, False in caso contrario.

Sample Input Sample Output
20Millions! True
30,20,10 False
FGPE_2020? True
