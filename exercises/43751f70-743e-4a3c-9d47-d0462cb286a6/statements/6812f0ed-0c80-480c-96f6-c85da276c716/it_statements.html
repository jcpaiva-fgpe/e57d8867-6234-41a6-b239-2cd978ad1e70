Gioco di scatole di cartone

Sei tornato da un negozio di scarpe. Dopo aver messo via tutte le scarpe, ti rimangono molte scatole di cartone, che vuoi conservare per un uso successivo. Quindi prendi una borsa e ci metti dentro tutte le altre. In tal modo, ti rendi conto che ci sono vari modi per annidare le scatole, con tutte le scatole viste come uguali tra di loro. Se usiamo una coppia di parentesi corrispondente per rappresentare una scatola, le combinazioni possono essere:
Per 1 scatola, c'è un solo modo:
[] <- una scatola
Per 2 scatole, c'è un modo:
[[]] <- una scatola in un'altra
Per 3 scatole, c'è un modo:
[[[]]] <- 3 scatole nidificate stile bamboline russe
[[] []] <- 2 scatole una accanto all'altra, all'interno della terza
Per 4 scatole, quattro:
[[] [] []]
[[[]] []]
[[[] []]]
[[[[]]]]
Nota che poiché tutte le caselle sono identiche, le due scatole a 4 caselle [[[]] []] e [[] []]]] rappresentano la stessa configurazione. È facile vedere che ogni configurazione per n scatole rappresenta un albero con radice a n nodi, dove un box è un nodo albero, e un box con il suo contenuto forma un sottoalbero. La scatola più esterna è la radice dell'albero. Il numero di configurazioni per dato n è dato da OEIS A81. Attività Scrivere un programma che, quando viene fornito N, enumeri tutti i modi di nidificare n riquadri. Dovresti usare la notazione tra parentesi sopra. Questa attività richiede solo l'enumerazione degli alberi; per il conteggio delle soluzioni senza enumerazione, quella pagina OEIS elenca varie formule, ma questo non è incoraggiato da questa attività, soprattutto se l'implementazione aumenterebbe in modo significativo la dimensione del codice.
Suggerimenti Puoi scrivere 3 funzioni: • Boxes () • Boxchain () • Replace_brachets ()

Input

La prima riga di input contiene un numero intero N nei modi di nidificare n riquadri.

Output

L'output mostra N righe, ciascuna con un modo per annidare n caselle.
