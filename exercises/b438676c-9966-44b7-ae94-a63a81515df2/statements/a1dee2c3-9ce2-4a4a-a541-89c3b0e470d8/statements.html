Array with multiple of 3

Your task is to write a short program that takes in input an array A composed by N numbers and gives in output an array B that contains only values numbers multiple of 3.

Tips:

A%x Given the number A, if it returns 0, x is multiple of A

Input

The first line contains a number N that is the expected numbers to take in input.
The second line contains all the N numbers divided by space “ “ or “EMPTY” if the array is empty.

Output

The output contains the array B.

Sample Input Sample Output
3
12 4 34		 12
4
12 6 44 65		 12 6
2
2 4		 EMPTY
