Draw an Histogram

Your task is to print an histogram of all elements of an array of integers A of size N, given as input. To write an histogram, take an element from array and use "*" to represent it. (eg. 3 = ***)

Input

The first line of input contains an integer N (N <= 10), the size of array A.
The second line of input contains the values of array A.

Output

Display N lines which contains the symbol "*" for each element.

Sample Input Sample Output
3
5 9 2 		*****
*********
**
4
1 3 2 6 		*
***
**
******
2
8 10 15 16 		Wrong input
