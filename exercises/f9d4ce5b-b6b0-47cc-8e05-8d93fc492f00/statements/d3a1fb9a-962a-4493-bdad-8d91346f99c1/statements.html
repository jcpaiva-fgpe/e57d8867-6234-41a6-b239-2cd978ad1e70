Number Comparison

Write a program that takes in input two numbers A and B and compare them.
If A is major than B the output will be “>”.
If A is minor than B the output will be “<”.
If A is equal than B the output will be “==”.

Input

The first line contains the number A.
The first line contains the number B.

Output

The first line contains the comparison.

Sample Input Sample Output
4
5		 <
45
1		 >
22
22		 ==
