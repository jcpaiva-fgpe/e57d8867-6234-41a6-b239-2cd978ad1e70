Volume of a Spherical Shell

The volume of a spherical shell is the difference between the enclosed volume of the outer sphere and the enclosed volume of the inner sphere:

Your task is to write a function that takes r1 and r2 as arguments and returns the volume of a spherical shell rounded to the nearest thousandth.

Notes

The inputs are always positive numbers. r1 could be the inner radius or the outer radius, don't return a negative number.

Input

The first line of input contains the radius r1 and r2.

Output

The first line displays the volume of a spherical shell rounded to the nearest thousandth.
Sample Input Sample Output
3 3 0
7 2 1403.245
3 800 2144660471.753
