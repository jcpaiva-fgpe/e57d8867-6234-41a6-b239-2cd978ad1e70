Moran Numbers

Un numero Harshad è un numero divisibile per la somma delle sue cifre. Ad esempio, 132 è divisibile per 6 (1 + 3 + 2). Un sottoinsieme dei numeri Harshad sono i numeri Moran. I numeri Moran danno un numero primo se divisi per la somma delle loro cifre. Ad esempio, 133 diviso 7 (1 + 3 + 3) restituisce 19, un numero primo.

Il tuo compito è scrivere una funzione che accetta un numero e restituisce "M" se il numero è un numero Moran, "H" se è un numero Harshad (non Moran) o "Neither".

Input

La prima riga di input contiene un numero.

Output

La prima riga contiene "M" se il numero è un numero Moran, "H" se è un numero Harshad (non Moran) o "Neither".

Esempio di input Esempio di output
132 H
133 M
134 Neither
