Substring between two numbers

Write a program, enter a non-empty string S and two non-negative integers from the keyboard A, B of value not greater than the length of S and A <= B, display the substring of S between A and B (extremes included).
Validation of S, A and B is required.

Input

The first line contains the string S.
The second line contains two non-negative integers (A<=B).

Output

The first line contains substring S.
