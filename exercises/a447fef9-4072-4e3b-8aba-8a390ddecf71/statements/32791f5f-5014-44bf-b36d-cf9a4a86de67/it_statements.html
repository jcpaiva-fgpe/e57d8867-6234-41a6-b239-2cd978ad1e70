Media di esecuzione guadagnata

Il tuo compito è scrivere una funzione che restituisca una Earned Run Average (ERA).
Una ERA viene calcolato moltiplicando 9 per il quoziente di Earned Runs consentiti diviso per ip Innings Pitched.
Nelle statistiche del baseball, gli inning sono rappresentati con una parte frazionaria di. 1 (1/3) o. 2 (2/3) per rappresentare il numero di out in un inning.
Un numero intero o un numero con una parte frazionaria di. 0 rappresenta un inning completo con tre out.
Controlla la scheda Risorse per una spiegazione più approfondita.

Input

La prima riga di input contiene il quoziente di Pitched guadagnati consentiti e Innings Pitched separati da uno spazio.

Output

La prima riga contiene la Earned Run Average (ERA).

Sample Input Sample Output
22 99 2.00
23 99.1 2.08
24 99.2 2.17
