Higher average

Write a program that, after storing five integer numbers read from the keyboard in a vector, displays the arithmetic mean m and the list of vector elements greater than m.

Input

The first line contains the 5 numbers divided by space.

Output

The first line contains the arithmetic mean.
The second line contains the vector of elements greater than m.

Sample Input Sample Output
3 34 45 34 12 25,60
34 45 34
33 44 55 66 77 55,00
66 77
1 10 100 1000 10000 2222,20
10000
