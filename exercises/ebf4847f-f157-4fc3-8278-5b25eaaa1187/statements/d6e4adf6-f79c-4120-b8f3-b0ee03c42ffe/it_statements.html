Equazioni di prima elementare

Il tuo compito è scrivere un breve programma per risolvere le equazioni di primo grado. Un'equazione di primo grado è:

ax + b = 0

Dato in input A e B, risolvi la FGE.

Suggerimenti:

x = -b ⁄ a

Input

L'input dovrebbe essere formattato in questo modo "A B". Dove A è il valore del primo grado e B è il termine costante.

Output

L'output dovrebbe essere formattato come questa "X". La prima riga contiene il termine del risultato. Se è impossibile, dovrebbe restituire "IMPOSSIBLE"

Esempio di input Esempio di output
64 8 - 0,125
6 36 - 6
0 24 IMPOSSIBLE
