First Grade Equations

Your task is to write a short program to resolve First Grade Equations. A first grade equation is:

ax + b = 0

Given in input A and B, solve the FGE.

Tips:

x = -b⁄a

Input

The input should be formatted like this “A B”. Where A is the first grade value, and B is the constant term.

Output

The output should be formatted like this “X”. The first line contains the result term. If it is impossible it should return “IMPOSSIBLE”

Sample Input Sample Output
64 8 -0.125
6 36 -6
0 24 IMPOSSIBLE
