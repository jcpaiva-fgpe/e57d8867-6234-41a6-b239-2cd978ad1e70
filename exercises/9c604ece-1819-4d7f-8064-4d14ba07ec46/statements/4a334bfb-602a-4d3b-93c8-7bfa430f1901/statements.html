Hiding the Card Number

Your task is to write a program that takes a credit card number and only displays the last four characters. The rest of the card number must be replaced by ************.

Input

The first line of input contains the credit card number.

Output

The first line contains only the last four characters.

Sample Input Sample Output
1234123456785678 ************5678
8754456321113213 ************3213
35123413355523 **********5523
