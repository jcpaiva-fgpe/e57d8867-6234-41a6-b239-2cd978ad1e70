Codice esadecimale valido

Il tuo compito è scrivere una funzione che determini se una stringa è un codice esadecimale valido.

Un codice esadecimale deve iniziare con un cancelletto # ed è lungo esattamente 6 caratteri. Ogni carattere deve essere una cifra da 0 a 9 o un carattere alfabetico da A-F. Tutti i caratteri alfabetici possono essere maiuscoli o minuscoli.

Input

La prima riga di input contiene il codice esadecimale.

Output

La prima riga contiene se l'input è un codice esadecimale valido (vero o falso).

Esempio di input Esempio di output
# CD5C5C true
# CD5C58C false
