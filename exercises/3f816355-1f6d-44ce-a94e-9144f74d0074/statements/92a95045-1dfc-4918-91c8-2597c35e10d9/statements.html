Shuffle the name

Your task is to write a method that accepts a string (of a person's first and last name) and returns a string with the first and last name swapped.
Notes There will be exactly one space between the first and last name.

Input

The first line of input contains a person's first and last name.

Output

The first line contains the first and last name swapped.

Sample Input Sample Output
Rosie O'Donnell O'Donnell Rosie
Seymour Butts Butts Seymour
