Hex to Binary

Your task is to write a program that will take a HEX number and returns the binary equivalent (as a string).

Notes
The number will be always an 8-bit number.

Input

The first line of input contains the HEX number.

Output

The first line of output contains the binary equivalent.

Sample Input Sample Output
0xFF 11111111
0xAA 10101010
0xFA 11111010
