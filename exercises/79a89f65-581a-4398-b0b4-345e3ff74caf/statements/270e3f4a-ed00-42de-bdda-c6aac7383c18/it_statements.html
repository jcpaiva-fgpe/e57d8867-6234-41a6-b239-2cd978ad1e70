Da esadecimale a binario

Il tuo compito è scrivere un programma che prenda un numero HEX e restituisca l'equivalente binario (come una stringa).

Note
Il numero sarà sempre un numero a 8 bit.

Input

La prima riga di input contiene il numero HEX.

Output

La prima riga di output contiene l'equivalente binario.

Esempio di input Esempio di output
0xFF 11111111
0xAA 10101010
0xFA 11111010
