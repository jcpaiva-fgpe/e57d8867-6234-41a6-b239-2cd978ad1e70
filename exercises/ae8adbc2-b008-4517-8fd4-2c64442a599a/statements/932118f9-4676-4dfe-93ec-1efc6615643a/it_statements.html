Controlla le rime

Il tuo compito è scrivere una funzione check_rhymes che controlli se una parola W è in rima con ogni parola in un array R.
In questo caso, una parola è in rima con un'altra parola quando i loro ultimi 3 caratteri sono uguali.

Input

La prima riga di input contiene una parola W utilizzata per verificare se fa rima con parole nell'array R.
La seconda riga di input contiene alcune parole inserite nell'array R.

Output

L'output mostra le parole dell'array R che sono in rima con la parola W.

Sample Input Sample Output
word
air sword maximum lord		 sword lord
task
help job mask utility		 mask
axe
max phone netbook		 None
