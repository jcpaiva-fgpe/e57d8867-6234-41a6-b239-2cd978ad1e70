Check the rhymes

Your task is to write a function check_rhymes that check if a word W is in rhyme with each words in an array R.
In this case, a word is in rhyme with another word when their last 3 character are equals.

Input

The first line of input contains a word W used to check if its rhymes with words in array R.
The second line of input contains some words inserted in the array R.

Output

The output displays the words of the array R that are in rhymes with the word W.

Sample Input Sample Output
word
air sword maximum lord		 sword lord
task
help job mask utility		 mask
axe
max phone netbook		 None
