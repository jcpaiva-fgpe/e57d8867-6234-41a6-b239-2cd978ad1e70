Trova la bomba

Il tuo compito è scrivere un programma che trovi la parola "bomb" nella stringa data. Se c'è una bomba, restituisci "Duck !!!" altrimenti torna "There is no bomb, realx".

Input

La prima riga di input contiene una stringa S.

Output

La prima riga contiene "Duck !!!" o "There is no bomb, realx";.

Sample Input Sample Output
There is a bomb. Duck!!!
Hey, did you think ther is a bomb? Duck!!!
This goes boom!!!" There is no bomb, relax.
