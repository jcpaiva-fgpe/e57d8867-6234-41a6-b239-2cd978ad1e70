Utilizzo dell'operatore "&&"

JavaScript ha un operatore logico &&. L'operatore && accetta due valori booleani e restituisce true se entrambi i valori sono veri.
Considera a && b:
• a viene controllato se è vero o falso.
• Se a è falso, viene restituito false.
• b viene controllato se è vero o falso.
• Se b è falso, viene restituito false.
• In caso contrario, viene restituito true (poiché sia a che b sono quindi veri).
L'operatore && restituirà true solo per true && true. Il tuo compito è scrivere una funzione usando l'operatore &&.

Input

La prima riga di input contiene due valori booleani separati da uno spazio.

Output

La prima riga contiene "true" o "false".

Sample Input Sample Output
true false false
true true true
false false false
