Tripletta pitagorica

Il tuo compito è scrivere una funzione che convalidi se tre numeri interi formano una terzina pitagorica. La somma dei quadrati dei due numeri interi più piccoli deve essere uguale al quadrato del numero più grande per essere convalidata.

Input

La prima riga di input contiene tre numeri interi separati da uno spazio.

Output

La prima riga contiene "true" o "false".

Sample Input Sample Output
3 4 5 true
13 5 12 true
1 2 3 false
