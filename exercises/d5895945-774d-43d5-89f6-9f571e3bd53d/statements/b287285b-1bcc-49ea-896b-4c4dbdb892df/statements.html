Factorial

Your task is to write a short program that takes in input a number positive N (with N <= 10) and gives in output the N factorial.
If the number is wrong, the output is ERROR

TIPS
A factorial of a number N is the product of all the numbers between 1 and N.
N! = 1 · 2 · 3 · . . .· (N − 1) · N
REMEMBER 0! = 1

Input

The first line contains a number N.

Output

The output contains the factorial of N.

Sample Input Sample Output
10 3628800
13 ERROR
-13 ERROR
