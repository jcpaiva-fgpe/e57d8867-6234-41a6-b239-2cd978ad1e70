Cosa si nasconde tra la folla?

Una parola è in libertà e ora ha cercato di nascondersi tra una folla di lettere maiuscole!
Aiuta a scrivere una funzione per rilevare qual è la parola, conoscendo le seguenti regole:
• La parola desiderata è in minuscolo.
• La maggior parte di lettere è tutta in maiuscolo.
• Nota che la parola verrà distribuita tra le lettere casuali, ma le loro lettere rimangono nello stesso ordine.

Input

La prima riga di input contiene una stringa S.

Output

La prima riga contiene la parola nascosta.

Sample Input Sample Output
UcUNFYGaFYFYGtNUH cat
bEEFGBuFBRrHgUHlNFYaYr Burglar
YFemHUFBbezFBYzFBYLleGBYEFGBMENTment embezzlement
