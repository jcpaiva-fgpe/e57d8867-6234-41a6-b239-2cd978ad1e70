Average Matrix

Your task is to write a short program that takes in input a matrix A and its dimensions {NxM} and returns 2 vectors: Vector MR contains the average of each ROW.
Vector MC contains the average of each COLUMN.

Input

The first line contains two numbers N and M that is the dimension of the matrix A.
The other lines contains each ROW of the matrix A.

Output

The first line contains a vector of the row average.
The second line contains a vector of the column average.
