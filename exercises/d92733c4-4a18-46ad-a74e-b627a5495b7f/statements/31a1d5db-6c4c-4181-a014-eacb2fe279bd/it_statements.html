Il più grande elenco di numeri primi

Il tuo compito è scrivere una funzione che, dato un numero N, trovi la più grande lista di numeri primi che darà N dopo la somma della stessa.
Idea di base:
un numero può essere espresso come sommatoria dell'elenco più lungo di numeri primi solo quando è espresso solo con 2 e 3.

Input

La prima riga di input contiene il numero N.

Output

La prima riga contiene l'elenco più grande di numeri primi.

Sample Input Sample Output
7 2 2 3
23 2 2 2 2 2 2 2 2 2 2 3
