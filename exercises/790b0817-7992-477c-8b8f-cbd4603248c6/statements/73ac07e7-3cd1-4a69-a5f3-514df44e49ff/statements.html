Intersection

Say whether the two ranges of double numbers a-b and c-d included, intersect, (they have elements in common).

Input

The first line contains the coordinates of the first number (a b).
The second line contains the coordinates of the second number (c d).

Output

The first line contains INTERSECT or NOT INTERSECT.

Sample Input Sample Output
3 4
4 3		 INTERSECT
18 17
26 19		 NOT INTERSECT
3 4
3 7		 INTERSECT
