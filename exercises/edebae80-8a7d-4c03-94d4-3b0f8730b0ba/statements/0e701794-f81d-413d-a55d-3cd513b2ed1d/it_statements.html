Da binario a Decimale

Il tuo compito è scrivere un breve programma che prenda in input un numero binario e lo converta in un numero decimale. Se il bit inserito è errato l'output deve essere ERROR

Input

La prima riga contiene il numero totale di bit.
La seconda riga contiene il valore di ogni bit separato da spazio ““.

Output

L'output contiene il numero decimale.

Sample Input Sample Output
3
1 0 1		 5
4
1 0 1 0		 10
3
2 0 1		 ERROR
