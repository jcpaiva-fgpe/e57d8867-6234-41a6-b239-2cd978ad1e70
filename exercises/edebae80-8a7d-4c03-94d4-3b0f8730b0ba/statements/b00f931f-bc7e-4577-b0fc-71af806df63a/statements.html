Binary to Decimal

Bour task is to write a short program that takes in input a binary number and convert its to a decimal number. If the bit inserted is wrong, the output is ERROR.

Input

The first line contains the total bit number.
The second line contains the value of each bit separated by space “ “.

Output

The output contains the decimal number.

Sample Input Sample Output
3
1 0 1		 5
4
1 0 1 0		 10
3
2 0 1		 ERROR
