Spostamento di vettore

Scrivete un programma che prenda in input N numeri.
I numeri vengono memorizzati in un vettore.
Quindi dovresti seguire questi passaggi:
1. Spostamento a sinistra di una posizione del vettore. Ogni valore del vettore dovrebbe assumere il valore 2.
Spostamento di una posizione a destra del vettore.
Ogni valore del vettore dovrebbe assumere il valore di quello accanto al suo.
L'elemento N-1 assumerà valore 0.
Esempio: V = 1 2 3 4 Shift sinistro: V = 2 3 4 0 quello precedente al suo.
L'elemento dell'indice 0 assumerà il valore 0.
Esempio: V = 2 3 4 0 Spostamento a destra: V = 0 2 3 4

Input

La prima riga contiene un numero N, la dimensione dell'array.
La seconda riga contiene i valori dell'array.

Output

La prima riga contiene il vettore spostato a sinistra.
La seconda riga contiene il vettore spostato a destra.

Sample Input Sample Output
4
1 2 3 4		 2 3 4 0
0 2 3 4
3
23 45 34		 45 34 0
0 45 34
5
23 3 45 12 2		 3 45 12 2 0
0 3 45 12 2
