Digit Frequency

Given a string, S, consisting of alphabets and digits, your task is to write a program to find the frequency of each digit in the given string.

Input

The first line contains a string, num which is the given number.

Output

Print ten space-separated integers in a single line denoting the frequency of each digit from 0 to 9.

Sample Input Sample Output
a11472o5t6 0 2 1 0 1 1 1 1 0 0
lw4n88j12n1 0 2 1 0 1 0 0 0 2 0
