Frequenza delle cifre

Data una stringa, S, composta da alfabeti e cifre, il tuo compito è scrivere un programma per trovare la frequenza di ogni cifra nella stringa data.

Input

La prima riga contiene una stringa, num che è il numero specificato.

Output

Stampa dieci numeri interi separati da spazi in una singola riga che denotano la frequenza di ciascuna cifra da 0 a 9.

Sample Input Sample Output
a11472o5t6 0 2 1 0 1 1 1 1 0 0
lw4n88j12n1 0 2 1 0 1 0 0 0 2 0
