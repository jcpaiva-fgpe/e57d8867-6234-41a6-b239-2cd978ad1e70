Indirizzo MAC

Il tuo compito è inserire le parti mancanti del codice nella funzione checkMAC().
Questa funzione prende in input una stringa mac e controlla se si tratta di un indirizzo MAC. Un indirizzo MAC è composto da:
• 6 coppie esadecimali
• Ogni coppia è separata dal simbolo ':' Un esempio è: 30: 33: A4: F2: 55: 12

Input

La prima riga di input contiene una stringa mac

Output

L'output mostra True se la stringa mac è un indirizzo MAC, False in caso contrario.

Sample Input Sample Output
30:33:A4:F2:55:12:12 False
30:33:A4:F2:55:1D True
30:33:A4:F2:55::12 False
