MAC Address

Your task is to insert the missing parts of the code in the function checkMAC.
This function takes in input a string mac and check if it is a MAC address.
A MAC address is composed by: • 6 hexadecimal couples • Each couple is separated by the symbol ‘ : ‘ An example is: 30:33:A4:F2:55:12

Input

The first line of input contains a string mac

Output

The output displays True if the string mac is a MAC address, False otherwise.

Sample Input Sample Output
30:33:A4:F2:55:12:12 False
30:33:A4:F2:55:1D True
30:33:A4:F2:55::12 False
