Substring Substitution

Write a program that takes a string A (max 30 char) and a string B that is a substring of A. The program should replace the substring B in A with "*" every time.
Example:
A = abchdfffchdtlchd
B = chd
A (final) = ab***fff***tl***

Input

The first line contains a string A and B.

Output

The first line contains the string A corrected.

Sample Input Sample Output
abchdfffchdtlchd
chd		 ab***fff***tl***
hello world
or		 hello w**ld
hello
world!		 ERROR
