Media volumi

Il tuo compito è scrivere un breve programma che prenda in input N numeri che sono volumi (metri cubi) di N appartamenti e calcolare la media dei loro volumi.

Input

La prima riga contiene un numero N che è i numeri attesi da prendere in input.
La seconda riga contiene tutti gli N numeri divisi dallo spazio "".

Output

L'output contiene X la media.

Esempio di input Esempio di output
3
4.56 3.23 6.77 		4.85333
4
34,4 5,456 3,81 1,24 		11.2265
3
2.22 3.33 4.44 		3.33
