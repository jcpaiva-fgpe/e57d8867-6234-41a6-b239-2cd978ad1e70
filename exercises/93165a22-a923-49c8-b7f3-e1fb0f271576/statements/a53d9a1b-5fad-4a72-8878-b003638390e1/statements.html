Volumes Average

Your task is to write a short program that takes in input N numbers that are volumes (cubic meters) of N flats and calculate the average of their volumes.

Input

The first line contains a number N that is the expected numbers to take in input.
The second line contains all the N numbers divided by space “ “.

Output

The output contains X the average.
