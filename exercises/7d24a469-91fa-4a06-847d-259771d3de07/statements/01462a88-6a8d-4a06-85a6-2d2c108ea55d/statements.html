Parallelepiped

A programmer wrote this simple program. This is used to calculate area and volume of a parallelepiped. Unfortunately the program does not compile! Can you spot the bug and select the line where there is an error?

Input

The first line contains a len of parallelepipeds.
The second line contains the width of parallelepipeds.
The third line contains the height of parallelepipeds.

Output

The first line contains the total area.
The second line contains the volume.
