Correttore maiuscolo

Scrivi un programma che prenda una stringa A (max 100 caratteri) con caratteri maiuscoli e minuscoli e restituisca una stringa B dove ogni prima lettera di ogni parola è maiuscola e il resto è minuscolo.
Esempio:
ToDaY is sUNNy -> Today Is Sunny

Input

La prima riga contiene una stringa A.

Output

La prima riga contiene la stringa B corretta.

Sample Input Sample Output
tHiS iS a TEST This Is A Test
hello WORLD! Hello World!
toDaY iS reaLLY cOlD Today Is Really Cold
