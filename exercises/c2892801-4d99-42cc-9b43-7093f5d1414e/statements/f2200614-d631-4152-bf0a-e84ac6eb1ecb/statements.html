Uppercase Corrector

Write a program that takes a string A (max 100 char) with uppercase and lowercase characters and returns a string B where each first letter of each word is uppercase and the rest is lowercase.
Example: ToDaY is sUNNy -> Today Is Sunny

Input

The first line contains a string A.

Output

The first line contains the string B corrected.

Sample Input Sample Output
tHiS iS a TEST This Is A Test
hello WORLD! Hello World!
toDaY iS reaLLY cOlD Today Is Really Cold
