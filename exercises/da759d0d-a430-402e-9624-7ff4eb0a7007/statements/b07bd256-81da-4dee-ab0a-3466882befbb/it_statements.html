È un triangolo?

Scrivi un programma che inserisca i lati di un ipotetico triangolo, dì se esiste un tale triangolo.
(Ricorda che esiste un triangolo se ogni lato è inferiore alla somma degli altri due).

Input

La prima riga contiene tre numeri divisi dallo spazio ““.

Output

La prima riga dice se il triangolo esiste (SÌ o NO)

Sample Input Sample Output
33,3 34,3 34,4 YES
41,1 65,1 13,1 NO
55,5 56,7 76,8 YES
