Numeri pandigitali

Un numero pandigital contiene tutte le cifre (0-9) almeno una volta. Il tuo compito è scrivere una funzione che accetta un numero intero, restituendo true se l'intero è pandigitale e false in caso contrario.

Input

La prima riga di input contiene il numero intero.

Output

La prima riga contiene "true" o "false".

Sample Input Sample Output
98140723568910 true
90864523148909 false
90864523148909 false
