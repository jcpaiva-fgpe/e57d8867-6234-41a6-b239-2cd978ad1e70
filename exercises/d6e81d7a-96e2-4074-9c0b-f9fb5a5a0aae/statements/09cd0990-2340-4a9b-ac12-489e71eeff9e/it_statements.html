Equazioni di Secondo Grado

Il tuo compito è quello di scrivere un breve programma che risolva le Equazioni di Secondo Grado (SGE).
Una SGE è:
ax2 + bx + c = 0
Dato in input A, B e C, risolvere la SGE.

Input

L'input deve essere formattato "A B C". Dove A è il valore di secondo grado, B quello di primo e C la costante.

Output

L'output deve essere formattato "X1 X1".
Se l'equazione non è risolvibile, ritornare "IMPOSSIBLE".

Sample Input Sample Output
1 -1 -2 2 -1
34 5 10 IMPOSSIBLE
4 16 8 -0.585786 -3.41421
