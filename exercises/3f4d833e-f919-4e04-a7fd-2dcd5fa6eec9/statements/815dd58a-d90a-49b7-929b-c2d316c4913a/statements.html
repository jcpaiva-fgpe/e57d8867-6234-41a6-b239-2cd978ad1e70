Numbers Distinction

Write a program that, entered from the keyboard three whole numbers, prints one of the following on the screen: ALL if they are all equals TWO if two of them are equals NONE if no one is equal.

Input

The first line contains three numbers divided by space “ “.

Output

The first line displays the message.
(ALL, TWO, NONE)

Sample Input Sample Output
34 4 34 TWO
5 5 5 ALL
43 5 23 NONE
