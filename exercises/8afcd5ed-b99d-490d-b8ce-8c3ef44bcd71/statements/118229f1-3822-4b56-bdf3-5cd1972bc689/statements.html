Simple Geometry

Your task is to write two functions square() and circle(), each function should return:
- Square() returns the area of a square given a side X.
- Circle() returns the circumference of a given radius X.
Given an integer N (1 <= N <= 2) that select the function (1 = Circle and 2 = Square) and another integer M (the length of radius or side), display the area.

Input

The first line of input contains an integer N (1 <= N <= 2) to select the right function.
The second line of input contains an integer M.

Output

The first line displays "Wrong Input" if the constraints are not fulfilled, the calculated area otherwise.

Sample Input Sample Output
1
31 		3017.54
2
24 		576
5
123 		Wrong Input
